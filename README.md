# rollup-ts-template

让你简单的创建一个rollup 和 ts 的模板

## Includes

- [x] TypeScript
- [x] JavaScript
- [x] rollup

## Get started

### 1. 全局安装这个脚本|Install

> pnpm install @mekefly/templates -g

### 2. 执行|Run

> mekefly-templates fetch  rollup-ts-template  -N `packageName`

### 3. 做你爱做的事|Start your work

```shell
cd `packageName`

pnpm install

pnpm build
```
